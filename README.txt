# Blog do NhobaDev

Bem-vindo ao **Blog do NhobaDev**, um projeto desenvolvido para compartilhar conhecimentos, experiências e aprendizados na jornada de desenvolvimento Fullstack. Este blog foi construído com foco em boas práticas de desenvolvimento web, utilizando tecnologias modernas e responsivas.

## 📋 Descrição do Projeto

O Blog do NhobaDev é uma aplicação web estática que apresenta postagens sobre desenvolvimento Fullstack, organizadas em uma interface limpa e responsiva. O projeto foi desenvolvido com HTML, CSS e JavaScript, e é ideal para iniciantes que desejam aprender a criar blogs ou portfólios pessoais.

### Funcionalidades

- **Postagens em Destaque**: Uma seção dedicada para destacar a postagem mais relevante.
- **Listagem de Postagens**: Exibição de postagens recentes com links para páginas individuais.
- **Página Sobre**: Uma página dedicada para apresentar o autor e sua jornada no desenvolvimento.
- **Design Responsivo**: Layout adaptável para diferentes dispositivos (desktop, tablet e mobile).
- **Imagens Ampliáveis**: As imagens das postagens podem ser ampliadas ao clicar.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação do conteúdo.
- **CSS3**: Estilização e design responsivo.
- **JavaScript (opcional)**: Para futuras interações dinâmicas.
- **GitHub Pages**: Hospedagem do projeto (opcional).

## 🚀 Como Executar o Projeto

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/seu-usuario/Projeto1-Blog.git
   ```

2. **Abra o projeto no navegador**:
   - Navegue até o diretório do projeto.
   - Abra o arquivo `index.html` no navegador.

3. **Personalize o conteúdo**:
   - Edite os arquivos HTML para adicionar suas próprias postagens.
   - Substitua as imagens no diretório `image/` conforme necessário.
